SPAD Simulator — Physics & Code

A module-by-module reference mapping semiconductor device physics to the Python codebase.

Overview

This simulator models a 1D InGaAs/InP Separate Absorption, Charge, and Multiplication (SACM) SPAD. Given a layer stack, doping profile, and applied bias, it solves for:

  1. Electrostatic potential $\phi(x)$ and field $E(x)$ via Poisson's equation
  2. Impact ionization coefficients $\alpha(E)$, $\beta(E)$
  3. Avalanche trigger probabilities $P_e(x)$, $P_h(x)$
  4. Breakdown voltage $V_{BR}$, dark count rate (DCR), photon detection probability (PDP)
  5. Optional: Monte Carlo avalanche dynamics and timing jitter
Unit system: CGS-SI hybrid — lengths in cm, energies in eV, carrier densities in cm$^{-3}$, fields in V/cm. All physical constants in src/core/constants.py.

Architecture

Device (layers, materials, doping, grid) │ └─ SPADSimulator (orchestrator) ├── PoissonService ──→ φ(x), E(x) ├── IonizationCoefficients ──→ α(x), β(x) ├── TriggerSolver ──→ P_e(x), P_h(x) ├── DarkCurrentModel ──→ DCR ├── PDPModel ──→ PDP(λ) ├── DriftDiffusion + MonteCarloSimulator ──→ carrier transport └── SelfConsistentLoop ──→ PIC feedback

Module 2: Poisson Solver

Given a device and applied bias, solves for $\phi(x)$ and $E(x)$.

2.1 The Equation

$$\frac{d}{dx}\left[\varepsilon(x) \frac{d\phi}{dx}\right] = -\rho(x)$$ $$\rho(x) = q\big[p(x) - n(x) + N_D(x) - N_A(x)\big]$$

Carrier densities couple to potential via Boltzmann statistics:

$$n(x) = n_i \, e^{(\phi - \phi_n)/V_T}, \qquad p(x) = n_i \, e^{(\phi_p - \phi)/V_T}$$

where $V_T = k_B T / q$ is the thermal voltage.

2.2 Boundary Conditions

src/poisson/solver.py:43-51 — Dirichlet BCs at contacts:

$$\phi(0) = -V_T \ln\!\left(\frac{N_A}{n_i}\right), \qquad \phi(L) = V_{\text{bias}} + V_T \ln\!\left(\frac{N_D}{n_i}\right)$$

2.3 Nonlinear Newton-Raphson

The equation is nonlinear (exponential coupling). The solver iterates:

  1. Compute residual $F(\phi^k)$ — the discretized Poisson equation
  2. Compute Jacobian $J = \partial F / \partial \phi$ — tridiagonal matrix
  3. Solve $J \cdot \delta = -F$ via banded solver
  4. Damped update: $\phi^{k+1} = \phi^k + \alpha \cdot \delta$ with $\alpha = 0.5$ initially
  5. Backtracking line search: halve $\alpha$ until residual decreases

2.4 Discretized Residual

src/poisson/solver.py:78-91

$$F_i = \frac{\varepsilon_{i+\frac{1}{2}}(\phi_{i+1} - \phi_i) - \varepsilon_{i-\frac{1}{2}}(\phi_i - \phi_{i-1})}{dx^2} + \rho_i$$

where $\varepsilon_{i+\frac{1}{2}} = \frac{1}{2}(\varepsilon_i + \varepsilon_{i+1})$ — average at cell interface for heterojunctions.

2.5 Jacobian (Tridiagonal)

src/poisson/solver.py:93-106

$$\frac{\partial \rho}{\partial \phi} = -\frac{q}{V_T}(n + p)$$

Diagonal elements: $d_i = -(\varepsilon_{i-\frac{1}{2}} + \varepsilon_{i+\frac{1}{2}})/dx^2 + \partial\rho/\partial\phi_i$

2.6 Voltage Ramp

src/poisson/service.py:28-45 — Ramps from 0 to $V_{\text{bias}}$ in 2V steps to ensure convergence.

Module 3: Impact Ionization

When a carrier gains enough energy from the field, it can create an electron-hole pair via impact ionization.

3.1 What α and β Mean

CoefficientCarrierMeaning
$\alpha(E)$ElectronIonization probability per cm at field $E$
$\beta(E)$HoleIonization probability per cm at field $E$

The ratio $k = \beta/\alpha$ determines excess noise. For InP: $k \approx 0.4$–$0.5$.

3.2 Okuto-Crowell Model (Default)

src/avalanche/ionization.py:20-53

$$\alpha(E, T) = \frac{qE}{E_{th}} \exp\!\left\{A - \sqrt{A^2 + \left(\frac{E_{th}}{qE\lambda}\right)^2}\right\}$$ $$A = 0.217 \left(\frac{E_{th}}{E_R}\right)^{1.14}$$

Temperature dependence via optical phonon scattering:

$$\lambda(T) = \lambda_0 \cdot \tanh\!\left(\frac{\hbar\omega}{2k_BT}\right)$$

3.3 Chynoweth Model

src/avalanche/ionization.py:56-86 — Simplest: two-parameter exponential.

$$\alpha(E) = A \cdot \exp\!\left(-\frac{B}{E}\right), \qquad B = \frac{E_{th}}{\lambda}$$

3.4 Wolff Model

src/avalanche/ionization.py:89-115 — Semi-physical with scattering competition.

$$\alpha(E) = \frac{E}{E_{th}} \exp\!\left(-\frac{\pi}{2}\left(\frac{E_{th}}{E\lambda}\right)^2\right)$$

3.5 Dead Space

src/avalanche/ionization.py:169-214 — A newly born carrier needs distance $d_{\text{dead}}$ to gain threshold energy:

$$d_{\text{dead}} = \frac{E_{th}}{q|E|}$$

Inside the dead zone: $\alpha_{\text{eff}} = 0$. Outside: $\alpha_{\text{eff}} = \alpha$.

Module 4: Trigger Probability

Given $\alpha(x)$ and $\beta(x)$, what's the probability a carrier triggers a self-sustaining avalanche?

4.1 McIntyre Formalism

src/avalanche/trigger.py

Electron injection at position $x$:

$$\mu(x) = \exp\!\left(\int_x^W [\alpha(s) - \beta(s)]\, ds\right)$$ $$M(x) = \frac{\mu(x)}{1 - \int_x^W \beta(s)\,\mu(s)\, ds}$$ $$P_e(x) = 1 - \frac{1}{M(x)}$$

4.2 What are $\mu$ and $M$?

SymbolNamePhysical Meaning
$\mu(x)$Net multiplication integralAccounts for the asymmetry between electron and hole ionization. When $\alpha > \beta$, electrons multiply faster than holes — $\mu$ captures this directional bias. It is the ratio of electron-initiated gain to the "bare" gain without hole feedback.
$M(x)$Mean avalanche gainThe average number of electron-hole pairs produced by one carrier injected at $x$. When $M \to \infty$, the denominator $1 - \int \beta\mu\,ds \to 0$ — this is the positive feedback loop: holes create electrons which create more holes. Divergence of $M$ is the mathematical signature of breakdown.

The denominator $1 - \int_x^W \beta(s)\mu(s)\,ds$ represents the positive feedback: holes traveling backward create electrons, which create more holes. When it approaches zero → gain diverges → breakdown.

Hole injection is symmetric: swap $\alpha \leftrightarrow \beta$ and integrate from $0$ to $x$.

4.2 Code Implementation

Lines 31-44 — Reverse cumulative sum for right-to-left integration:

avg_diff = (ab_diff[:-1] + ab_diff[1:]) / 2.0   # trapezoidal average
exponents = avg_diff * dx                          # (α−β)·dx per cell
cum_exponents = np.concatenate([[0.0], np.cumsum(exponents[::-1])[::-1]])
mu = np.exp(cum_exponents)                         # μ(x)

4.3 Combined Trigger

src/core/physics_helpers.py:7-9

$$P_{\text{tr}} = P_e + P_h - P_e \cdot P_h \qquad \text{(independent events)}$$

Module 5: Breakdown Detection

Finds the bias voltage where avalanche becomes self-sustaining.

5.1 Current Criterion

src/avalanche/breakdown.py:45-55 + src/simulator/breakdown.py

Breakdown is detected when the total multiplied current exceeds a threshold:

$$I_{\text{total}} = I_{\text{primary}} \times M > I_{\text{threshold}}$$

where $I_{\text{primary}} = A_{\text{det}} \int J_{\text{dark}}(x)\, dx$ is the unmultiplied dark current and $M$ is the avalanche gain from the maximum combined trigger probability.

5.2 Avalanche Gain

src/core/physics_helpers.py:12-14

$$M = \frac{1}{1 - P_{\text{tr,max}} + 10^{-15}} \leq 10^4$$

5.3 Sweep Algorithm

src/avalanche/breakdown.py:58-94 — Sweeps $V$ from $V_{\text{start}}$ to $V_{\text{max}}$ in steps of $V_{\text{step}}$:

  1. Solve Poisson at each bias → get $E(x)$
  2. Compute $\alpha(E)$, $\beta(E)$
  3. Compute trigger probabilities $P_e$, $P_h$
  4. Check criterion → first hit is $V_{BR}$

Module 6: Dark Current

Current flowing without illumination — determines the dark count rate (DCR).

6.1 Three Components

src/avalanche/dark_current.py + src/avalanche/current.py

ComponentFormulaPhysics
SRH$J_{\text{SRH}} = q \cdot \dfrac{n_i}{2\tau}$Thermal generation via mid-gap traps
BTBT$J_{\text{BTBT}} = A F^2 e^{-B E_g^{3/2}/F}$Band-to-band tunneling (Kane)
TAT$J_{\text{TAT}} = \dfrac{A F^2 N_T e^{-(B_1 E_{B1}^{3/2} + B_2 E_{B2}^{3/2})/F}}{N_v e^{-B_1 E_{B1}^{3/2}/F} + N_c e^{-B_2 E_{B2}^{3/2}/F}}$Trap-assisted tunneling (Hurkx)

6.2 DCR Calculation

$$\text{DCR} = A_{\text{det}} \int G_{\text{total}}(x) \cdot P_{\text{trigger}}(x)\, dx$$

Only carriers that trigger count — generation in the dead zone doesn't contribute.

Module 7: PDP & Photocurrent

Photon detection probability — the probability that an incident photon produces a detectable avalanche.

7.1 Beer-Lambert Absorption

src/avalanche/pdp.py

$$\text{PDP}(\lambda, V) = (1 - r) \cdot T_{\text{dead}}(\lambda) \int \alpha(\lambda) \, e^{-\alpha(\lambda)(x - x_{\text{abs}})} \cdot P_{\text{trigger}}(x)\, dx$$

7.2 Components

  • Reflectivity: $(1 - r)$ — typically $r \approx 0.1$ (10% surface reflection)
  • Dead-zone transmission: $T_{\text{dead}} = \prod_i e^{-\alpha_i d_i}$ — absorption in non-multiplying layers
  • Absorption integral: $\int \alpha e^{-\alpha x} P(x) dx$ — weighted by trigger probability

7.3 Photocurrent

src/avalanche/pdp.py:69-82 — Generation rate from incident photon flux $\Phi$:

$$G_{\text{opt}}(x) = \Phi \cdot \alpha \cdot e^{-\alpha(x - x_{\text{start}})}$$ $$J_{\text{ph}} = q \cdot G_{\text{opt}}$$

Module 8: Carrier Transport

Models how individual carriers move through the device under field and diffusion.

8.1 Drift-Diffusion Step

src/transport/drift_diffusion.py

$$x_{\text{new}} = x + v_d \, dt + \sqrt{2D\, dt} \cdot \xi$$

where $\xi \sim \mathcal{N}(0,1)$ is a Gaussian random variable (Wiener process).

8.2 Drift Velocity (with Saturation)

$$v_d = \frac{\mu E}{1 + |E|/E_{\text{sat}}}$$

At low field: $v_d \approx \mu E$ (Ohmic). At high field: $v_d \to \mu E_{\text{sat}} = v_{\text{sat}}$ (velocity saturation).

8.3 Diffusion Coefficient

$$D_0 = \mu V_T \qquad \text{(Einstein relation)}$$

Enhanced at high field:

$$D = D_0 \left(1 + 2\left(\frac{|E|}{v_{\text{sat}}}\right)^2\right)$$

8.4 Carrier Object

src/transport/carrier.py — Tracks position, type, dead space, and alive status.

Module 9: Monte Carlo Avalanche

Simulates the stochastic avalanche build-up carrier by carrier.

9.1 Algorithm

src/transport/monte_carlo.py

  1. Inject seed carrier at $x_0$
  2. Drift-diffuse each carrier (drift-diffusion step)
  3. After dead space: sample ionization — $P_{\text{ion}} = 1 - e^{-\alpha \cdot dx}$
  4. On ionization: spawn new electron-hole pair at same position
  5. Stop when carrier count $\geq N_{\text{threshold}}$ → records detection time $t_{\text{detect}}$

9.2 Ensemble Statistics

Run $N_{\text{sim}}$ independent histories. Extract:

  • BrP = $N_{\text{avalanches}} / N_{\text{sim}}$ — breakdown probability
  • Timing jitter: $\sigma_t = \text{std}(t_{\text{detect}})$, FWHM, percentiles

9.3 Jitter Extraction

src/transport/jitter.py — Mean build-up time, standard deviation, FWHM from histogram.

Module 10: Self-Consistent PIC Loop

Particle-in-cell simulation coupling carrier dynamics, Poisson field, and bias circuit.

10.1 Components

ClassFileRole
ParticleMeshself_consistent/particle_mesh.pyCIC charge deposition + field interpolation
CircuitSolverself_consistent/circuit.py$V_{\text{bias}} = V_{\text{supply}} - I \cdot R_q - Q/C_{\text{spad}}$
SelfConsistentLoopself_consistent/loop.pyIterates: Poisson → field → transport → ionization → deposit → circuit

10.2 PIC Iteration

  1. Deposit carrier charges onto grid (CIC weighting)
  2. Add external charge to Poisson → solve for updated $\phi$, $E$
  3. Move carriers using local field
  4. Sample ionization → create new carriers
  5. Update circuit: $V_{\text{bias}}^{k+1} = V_{\text{supply}} - I^k R_q$
  6. Repeat until quenching (current drops below threshold)

10.3 Quenching

The quench resistor $R_q$ provides negative feedback: as avalanche current increases, the voltage drop across $R_q$ reduces the effective bias below $V_{BR}$, stopping the avalanche.
